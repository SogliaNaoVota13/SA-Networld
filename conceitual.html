Modelo Conceitual (DER)

Modelo do exercício




O que é?

Um Modelo Conceitual (DER), ou Diagrama Entidade-Relacionamento, é uma ferramenta visual usada para planejar e organizar os dados de um sistema antes de construir o banco de dados. Ele mostra de forma simplificada: Entidades: Representam coisas ou objetos do mundo real que precisam ser armazenados, como clientes, produtos ou pedidos. Relacionamentos: Indicam como essas entidades estão conectadas entre si, como "um cliente pode fazer vários pedidos". Atributos: São as características de cada entidade, como nome, idade ou preço. É como um rascunho que ajuda a entender o que será necessário guardar e como os dados se relacionam, antes de passar para a implementação técnica.


Dicionario de Dados

O que é?

Um dicionário de dados é um documento ou ferramenta que descreve de forma detalhada todos os elementos de dados usados em um sistema ou banco de dados. Ele funciona como um guia para entender e padronizar os dados. O objetivo é garantir que todos os envolvidos no projeto (analistas, programadores e usuários) entendam os dados da mesma forma, evitando erros e inconsistências.
