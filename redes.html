Equipamentos aplicados em Redes

Roteador

Função: Conectam diferentes redes, sejam elas locais ou remotas. Como funciona: Roteiam pacotes de dados entre diferentes redes, analisando o endereço IP de destino. Vantagens: Permitem a criação de redes maiores e mais complexas, além de oferecer recursos como NAT e firewall

Cabo de Rede LAN

Função: Conecta duas redes com protocolos diferentes. Como funciona: Traduz os protocolos de uma rede para os protocolos da outra rede. Exemplo: Um gateway residencial converte os protocolos da sua rede doméstica para os protocolos da internet.

Repetidor

Função: Ampliam o alcance de um sinal de rede, permitindo a conexão de dispositivos a distâncias maiores. Como funciona: Recebem um sinal enfraquecido e o retransmitem com maior potência. Vantagens: Aumentam o alcance da rede.

Switches

Função: Conectam diversos dispositivos em uma rede local (LAN), criando canais de comunicação dedicados entre eles. Como funciona: Ao receber um pacote de dados, o switch analisa o endereço de destino e encaminha o pacote apenas para o dispositivo correto, evitando o envio para todos os dispositivos conectados.


Conceitos de:

Servidor

Um servidor é um computador ou dispositivo que fornece serviços a outros dispositivos em uma rede. Ele armazena e gerencia dados, aplicativos e recursos que podem ser acessados por outros computadores, conhecidos como clientes

HTTP

HTTPS é um protocolo de comunicação seguro utilizado na internet para transmitir dados entre um servidor web e um navegador. Ele é uma versão do HTTP (Hypertext Transfer Protocol) que utiliza criptografia para proteger as informações trocadas, como senhas e dados de cartão de crédito.

DNS

O DHCP é um protocolo de rede que atribui automaticamente endereços IP e outras informações de configuração de rede a dispositivos conectados a uma rede. Isso facilita a configuração de redes, pois os dispositivos não precisam ser configurados manualmente com um endereço IP.

DHCP

O DNS é um sistema de nomenclatura hierárquico que traduz nomes de domínio (como www.exemplo.com) em endereços IP numéricos (como 192.168.1.1), que são utilizados pelos computadores para localizar e se conectar a outros dispositivos na internet.


Pinagem cabo de rede, Modelo OSI e TCP/IP

Pinagem

Pinagem padrão T568A

  • Branco/Verde – Transmissão de dados (+)
  • Verde – Transmissão de dados (-)
  • Branco/Laranja – Recepção de dados (+)
  • Azul – Terra, ou sinal de controle
  • Branco/Azul – Terra, ou sinal de controle
  • Laranja – Recepção de dados (-)
  • Branco/Marrom – Não utilizado (pode ser usado em instalações mais avançadas)
  • Marrom – Não utilizado (pode ser usado em instalações mais avançadas)

    • Modelo OSI

    Modelo conceitual usado para entender a rede de computadores

  • Camada 1 (Física): Transmissão de sinais elétricos e físicos.
  • Camada 2 (Link de Dados): Controle de erros e controle de fluxo, endereçamento MAC.
  • Camada 3 (Rede): Roteamento e endereçamento IP.
  • Camada 4 (Transporte): Controle de fluxo de dados, confiabilidade (TCP, UDP).
  • Camada 5 (Sessão): Controle das sessões de comunicação entre os sistemas.
  • Camada 6 (Apresentação): Formatação e criptografia de dados.
  • Camada 7 (Aplicação): Interação direta com o usuário, protocolos de aplicação como HTTP, FTP.

    • Conceito de TCP/IP

    O modelo TCP/IP é um conjunto de protocolos utilizado para a comunicação na internet e redes.

  • Camada 1 (Link de Rede): Equivalente às camadas físicas e de enlace do OSI, define como os dispositivos se comunicam fisicamente.
  • Camada 2 (Internet): Equivalente à camada de rede do OSI, inclui o protocolo IP, responsável pelo endereçamento e roteamento de pacotes.
  • Camada 3 (Transporte): Semelhante à camada de transporte do OSI, inclui protocolos como TCP e UDP para controle de fluxo e garantia de entrega.
  • Camada 4 (Aplicação): Combina as camadas de sessão, apresentação e aplicação do modelo OSI. Protocolos como HTTP, FTP e SMTP operam aqui.
