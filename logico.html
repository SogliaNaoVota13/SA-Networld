Modelo Lógico (MER)

Modelo do exercício




O que é?

O Modelo Lógico (MER), ou Modelo Entidade-Relacionamento, é uma representação mais detalhada e técnica de como os dados serão organizados no banco de dados. Ele é uma evolução do modelo conceitual (DER) e serve como um "projeto técnico" antes da implementação. Características principais: Entidades viram tabelas: Cada entidade identificada no modelo conceitual se transforma em uma tabela do banco. Relacionamentos viram chaves estrangeiras: As conexões entre entidades são representadas por colunas que ligam tabelas, chamadas de chaves estrangeiras. Atributos viram colunas: As características das entidades (atributos) se tornam colunas dentro das tabelas. Regras específicas: Define os tipos de dados (número, texto, data) e restrições (chave primária, única, obrigatória).
